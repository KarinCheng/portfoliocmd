Challenge

Openbare Bibliotheek Amsterdam (OBA) is a public library in Amsterdam. They have books for all ages. OBA has two separate websites for adults and children, but teens don't fit in either of them. So our task was to create a website for teens. This target group has different needs than kids and adults. For the project, it was important to remain consistent with OBA's visual design.

Solution

We created an outstanding visual design, because teens want to present themselves differently from other age groups. The design has a different color pallet, but the color and shape contrast remains. Through the whole app we created a responsive disclosure design to make it easy for the user to understand.

Structure

Discover a book

When the user opens the app, it will ask them to choose their age. This way OBA knows which website will suit them the best. The OBA teen is meant to be for kids who are between 12 and 15 years old. The most popular genres are listed at the top; this helps the user find a book quickly.When the user chooses a genre a list with all available books will appear.

Reserve a book

The user can reserve a book by tapping on the reserve button on the book detail screen. Then the user has to choose at what library they want to pick up the book. More information about the library will appear when they chose a library. At this screen they can confirm the reservation.
