99 Verhalen Voor Onderweg

  • Client:

    Personal Project

  • Category:

    Traveling

  • Product:

    Responsive story website

  • Project timeline:

    3 months

Challenge

Transavia is a Dutch low-cost airline. They want travelers to realise “low cost feel good.” The task was to create a website that would entertain travelers with 99 short stories on their way. The website should be easy transferable any readable on every device whenever the user wants. In order to design a new website, we had to create a well-made information structure for the stories and make the website looks and feel like it is part of Transavia.

Solution

We created a website where travelers can download a bundle of stories beforehand to read them in the plane without wifi. After the trip, they can look up all the stories they read and leave a comment on their favorite stories.

Before the trip
Users look up an interesting story bundle to read and downloads it.
During the trip
Users are able to read the stories offline in the plain.
After the trip
Users can look up the stories they read and leave a like or comment on their favorite stories.

View the Website
Structure

Discover Stories

To make the user feel more involved to our product, we divide the stories based on the final destination they are going to visit. The user can choose from seven destinations. Each destination has a bundle of stories based on the mood of the destination.

Time

Because every travel duration is different, we offer the option to read stories that would take about 60, 90, or 120 minutes.

Search option

If the user wants to read a specific mood or story they can use the search option. It allows the user to search moods or story names.

Reading Offline

It is most likely that the user would not be connected to the WiFi when they are traveling. So we provide an option to download the stories before hand. Users can read it whenever and wherever. All the downloaded stories will show up in download section in the menu.

Post Comment

Users can leave a comment on a story if they really liked it. This way the users would feel like they are in a community. They write their comment in the textbox and share it by pressing the call to action button.
